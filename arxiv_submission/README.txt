QDNU: Quantum Positive-Negative Neuron for EEG Analysis
=======================================================

arXiv Submission Package

Files included:
- main.tex         : LaTeX source for the paper
- figures/         : Directory containing figures
  - agate_circuit.png        : A-Gate quantum circuit structure
  - complexity_comparison.png : Classical vs quantum scaling

To compile:
